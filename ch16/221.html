Para configurar la función RSS Pan:

  1. Mantenga presionada la tecla MAYÚS y presione F4 • UTILIDAD.
  2. Presione PAGE repetidamente hasta que aparezca “RSSPan” encima de F2.
  3. Presione F2 (RSSPan); aparece la pantalla RSS PAN SETUP:
  4. Mueva el cursor hasta el parámetro Use EFFECT Board y gire el dial Time / Value para seleccionar la placa de efectos que le gustaría utilizar para producir el efecto RSS Pan. Puede seleccionar Desactivado (para no RSS Pan) o VS8F-2 A o B.

    5. Puede utilizar la placa de efectos B solo si su VS-2400CD tiene dos VS8F-2 instalados.

  5. Presione ENTER / YES para confirmar su selección.
  6. Si desea escuchar RSS Pan en su conector PHONES, encienda el interruptor PHONES.
  7. Para seleccionar el primer canal al que le gustaría aplicar la panorámica RSS. cursor hacia abajo hasta la línea superior en el cuadro USE CHANNEL LIST y gire el dial Time / Value para seleccionar la entrada deseada o el canal de pista.
  8. Para enviar señales de canales adicionales al efecto Panorámica RSS, coloque el cursor en las otras líneas en el cuadro Use CHANNEL LIST y gire el dial Time / Value para seleccionar el canal o canales deseados.

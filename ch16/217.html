Haga coincidir el parche con el proyecto

Cuando la frecuencia de muestreo de un proyecto es de 64 kHz o superior (consulte la página 92), los parches basados en los siguientes algoritmos se atenúan y no se pueden utilizar:

  • Reverberación
  • Gate Reverb
  • Codificador de voz2 (19)
  • Transformador de voz
  • Modelado de micrófono
  • Modelado de altavoces
  • Kit de herramientas de masterización

Seleccionar un parche de efectos

Para navegar a la pantalla PATCH SELECT de un efecto si aún no está allí:

  1. Mantenga presionada la tecla MAYÚS y presione F3 • EFECTO.
  2. Presione el botón F del procesador para mostrar su pantalla Vista de algoritmo.
  3. Presione F1 (PATCH).

Para seleccionar un parche de efecto en la pantalla PATCH SELECT:

  1. Presione F1 (PRESET) para saltar al primer parche de efecto predefinido en la lista de parches, o presione F2 (USER) para saltar al primer parche de usuario.
  2. Gire el dial Time / Value para seleccionar el parche deseado. Puede utilizar el dial para desplazarse por todos los programas predeterminados y de usuario si lo desea.
  3. Presione F5 (SELECT) para cargar el parche de efecto seleccionado.

Los Apéndices del VS-2400CD contienen una lista de programas de efectos que le indica si un programa está diseñado para usarse como efecto de bucle o de inserción. La lista también le indica si el parche tiene entradas estéreo, por lo que trata sus lados izquierdo y derecho de forma independiente, o si su entrada es mono.

El interruptor BYPASS se explica en la página 214. Presione F6 (EXIT) para regresar a la pantalla de visualización de algoritmos del procesador.

Edición de patches de efectos

Presione F3 (EDIT) en la pantalla Algorithm View de un procesador de efectos para ver la primera pantalla de edición para el parche de efecto seleccionado. En la siguiente ilustración, puede ver las dos pantallas de edición para el primer parche de efectos, P000 Reverb.

Cada algoritmo proporciona un conjunto de parámetros con los que puede cambiar el procesamiento de efectos que produce. Para conocer los parámetros de cada algoritmo, consulte los Apéndices del VS-2400CD. Cuando edita un parche, edita los valores del algoritmo (o algoritmos) en los que se basa el parche.

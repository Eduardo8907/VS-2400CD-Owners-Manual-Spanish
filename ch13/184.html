Borrar marcadores

Además de los dos métodos enumerados aquí, puede borrar un marcador en la pantalla Marcador del menú UTILIDAD, que se describe en "Edición de marcadores" a continuación.

Borrar un marcador usando ANTERIOR y SIGUIENTE

  1. Vaya al marcador que desea borrar usando ANTERIOR o SIGUIENTE (ver arriba)
  2. Mantenga pulsada CLEAR y pulse TAP.

Borrar un marcador en la ventana Borrar marcador

  1. Mantenga presionada la tecla MAYÚS y presione F6 • MARCADOR: aparece la ventana Localizar en marcador.
  2. Gire el dial de tiempo / valor para colocar una flecha a la izquierda del marcador deseado.
  3. Presione CLEAR — aparece la ventana Clear Marker
  4. Gire el dial de tiempo / valor para seleccionar otro marcador si lo desea.
  5. Presione ENTER / YES para borrar el marcador seleccionado
  6. Presione EXIT / NO para salir de la ventana Clear Marker.
  7. Presione EXIT / NO para cerrar la ventana Localizar al marcador.

Borrar todos los marcadores a la vez

Cuando borra todos los marcadores, todos los marcadores del proyecto se eliminan, incluidos todos los marcadores de pista de CD

  1. Mantenga presionados SHIFT y CLEAR y presione TAP. El VS-2400CD le pregunta si está seguro de que desea eliminar todos los marcadores del proyecto.
  2. Presione ENTER / YES para borrar todos los marcadores del proyecto, o EXIT / NO para cancelar la operación.

Editar marcadores

La pantalla Marcador del menú UTILIDAD proporciona herramientas para nombrar, borrar, restablecer o saltar a cualquier marcador.

  1. Mantenga presionada la tecla MAYÚS y presione F4 • UTILIDAD.
  2. Presione PAGE hasta que aparezca “MARKER” encima de F4.
  3. Presione F4 (MARCADOR).
  4. Presione ↑ o ↓ o gire el dial de tiempo / valor para seleccionar un marcador para editar. Usted puede:
    • Presione F1 (NAME) para cambiar el nombre (Página 71 ) del marcador seleccionado.
    • Presione F2 (CLEAR) para borrar el marcador seleccionado.
    • Presione F3 (GetNow) para restablecer el marcador a la posición actual de la línea actual.
    • Presione F4 (IR A) para mover la línea ahora a la posición del marcador seleccionado.
  5. Presione F6 (EXIT) cuando haya terminado para confirmar sus cambios.

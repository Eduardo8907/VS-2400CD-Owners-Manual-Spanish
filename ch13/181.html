Cambiar automáticamente al modo de localizador

Puede configurar el VS-2400CD para que permanezca en el modo Escena después de recuperar y almacenar escenas, o para que vuelva automáticamente al modo Localizador después de cualquier acción.

  1. Mantenga presionada la tecla MAYÚS y presione F4 • UTILIDAD.
  2. Si no ve "GLOBAL" encima de F2, presione PAGE hasta que lo vea.
  3. Presione F2 (GLOBAL)
  4. Si F1 (Param1) no está resaltado, presione F1 (Param1)
  5. Puede configurar el parámetro RETURN TO LOCATE Sw en:
    • Activado: para volver automáticamente al modo de localización después de almacenar o recuperar una escena.
    • Desactivado: para volver al modo de localización presionando manualmente LOCATE.
  6. Presione F6 (EXIT) para confirmar sus cambios.

Localizadores en modo seguro

Si prefiere trabajar con localizadores un poco más lenta y cuidadosamente, puede usar el modo seguro. El modo seguro le permite trabajar con localizadores en una ventana de LOCATOR que muestra el número y el nombre de cada localizador. También puede almacenar y borrar localizadores en modo seguro. Si bien esta es una forma más lenta de trabajar, brinda una mayor certeza de que tiene el localizador correcto.

Activar el modo seguro del localizador

  1. Mantenga presionada la tecla MAYÚS y presione F4 • UTILIDAD.
  2. Si "GLOBAL" no aparece encima de F2, presione PAGE hasta que aparezca.
  3. Presione F2 (GLOBAL).
  4. El parámetro LOCATOR / SCENE TYPE del menú UTILITY activa o desactiva el modo seguro; consulte la página 358. Su valor predeterminado es Rápido. Para activar el modo seguro, seleccione Seguro.

Recuperación de un localizador en modo seguro

  1. Presione LOCATOR: se abre la ventana LOCATOR.
  2. Gire el dial de tiempo / valor para desplazarse por la lista hasta que el localizador deseado esté a la vista.
  3. Ingrese el número de dos dígitos del localizador en el teclado; aparece una flecha a la izquierda del localizador en la lista y "¿Ubicar? aparece en la ventana.
  4. Presione ENTER / YES para recuperar el localizador seleccionado y cerrar la ventana, o simplemente presione ENTER / NO dos veces para salir de la ventana sin recuperar un localizador.

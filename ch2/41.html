Área del teclado

1 — Botón V-LINK

Presione el botón V-LINK — para que se encienda — para configurar el VS-2400CD para su uso con un sistema de presentación o edición de video. Cuando V-LINK está activo, el VS-2400CD y el equipo de video están sincronizados para una operación estrechamente integrada. Para obtener detalles sobre la sincronización, consulte el Capítulo 23.

MMC, MTC, esclava

2 — CD-RW • Botón MASTERING

Presione este botón para ver el menú CD-RW / MASTERING desde el cual puede realizar todas las operaciones relacionadas con el CD. Mantenga pulsado SHIFT y PULSE CD-RW • MASTERING para saltar a la sala de masterización del VS-2400CD, donde puede completar la mezcla maestra de un proyecto y crear un CD de audio. Consulte el Capítulo 26, a partir de la página 331.

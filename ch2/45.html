Controles básicos y herramientas de navegación

Esta área contiene controles de uso frecuente que forman parte de muchas operaciones del VS-2400CD.

1— Botones (Cursor) / ZOOM

Presione los botones de cursor,, o para mover el cursor de selección ( Página 65 ) alrededor de la pantalla del VS-2400CD.

Mantenga presionado SHIFT y presione el botón de flecha deseado para acercar y alejar los datos de la pista que se muestran en el área de la lista de reproducción de la pantalla de inicio (Página 115 ) y otras pantallas:

  • Presione → para aumentar la ampliación horizontal.
  • Presione ← para disminuir la ampliación horizontal.
  • Presione ↑ para aumentar la ampliación vertical.
  • Presione ↓ para disminuir la ampliación vertical.

2 — Botón SCRUB

Cuando necesite ubicar con precisión un momento en una pista, presione SCRUB para activar la función Scrub. Cuando SCRUB está encendido, puede girar el dial de tiempo / valor para mover la línea del ahora y escuchar una pequeña parte de la pista una y otra vez; esto le permite concentrarse en el momento que está buscando. Esto puede resultar muy útil al editar. Para apagar el fregado, presione SCRUB nuevamente para que se apague la luz. Consulte la página 178 para obtener más información.

3 — EDICIÓN MAESTRA • LCD↔VGA

Presione MASTER EDIT • LCD↔VGA para mostrar la pantalla MASTER EDIT VIEW (Página 201, Página 203 ). Cuando haya conectado un monitor VGA externo al VS2400CD, mantenga presionado SHIFT y presione MASTER EDIT • LCD↔VGA para seleccionar su pantalla principal. Consulte “La pantalla principal y la pantalla de información” en la página 381 para obtener más detalles.

4 — LOCATOR • Botón BANK

Presione el botón LOCATOR • BANK para que se encienda para almacenar y recuperar localizadores usando el teclado numérico (consulte “10 — Teclado numérico” en la página 43 ). Un localizador recuerda una posición de línea de ahora particular dentro de un proyecto; cuando recupera el localizador, la línea de ahora salta instantáneamente a esa posición de tiempo. Los localizadores se describen en detalle en “Localizadores” en la página 179.

Cada proyecto puede contener hasta 100 localizadores dispuestos en grupos o "bancos" de diez localizadores cada uno. Para seleccionar un banco localizador, mantenga presionada la tecla SHIFT, presione LOCATOR • BANK para que se ilumine y presione el número del banco deseado en el teclado numérico.

5 — Botón SCENE • BANK

Presione el botón SCENE • BANK para que se encienda para almacenar y recuperar escenas usando el teclado numérico (consulte “10 — Teclado numérico” en la página 43 ). Una escena almacena la configuración actual de su mezclador, lo que le permite guardar diferentes versiones de su proyecto para una recuperación instantánea. Las escenas se describen en detalle en “Escenas” en la página 139.

Cada proyecto puede contener hasta 100 escenas en grupos o “bancos” de diez escenas cada uno. Para seleccionar un banco de escenas, mantenga presionado SHIFT, presione SCENE • BANK para que se ilumine y presione el número del banco deseado en el teclado numérico.

6 — Botón FADER ASSIGN

Cuando presiona el botón FADER ASSIGN de modo que parpadee, puede presionar un número en el teclado numérico para asignar los faders al trabajo de enviar señales desde los canales individuales (Página 35) al bus Aux correspondiente. Además, puede presionar 9 • USER para asignarlos a otra configuración de su elección. Consulte la página 136 .

7 — NUMÉRICOS • Botón EXT SYNC

Presione NUMERICS para que se ilumine para ingresar valores numéricos para varios ajustes usando el teclado del VS-2400CD o un teclado ASCII conectado. Cuando NUMERICS está encendido, el área en la que puede escribir se selecciona en la pantalla. Consulte la página 72.

Mantenga presionado SHIFT y presione NUMERICS • EXT SYNC para que se encienda para sincronizar la reproducción del VS-2400CD con un dispositivo externo generador de MTC. Presiónelo de nuevo para que no se encienda cuando desee que el VS-2400CD proporcione la referencia de temporización maestra (este es el modo de funcionamiento normal del VS-2400CD). La sincronización se describe en el Capítulo 23.

Configuración de umbral y relación para un compresor

Si bien todos los parámetros de la pantalla DYN interactúan, Threshold y Ratio funcionan juntos como posiblemente los dos parámetros más fundamentales. A continuación, se muestran algunas pautas para configurar el umbral y la relación cuando se utiliza un compresor:

  • Para una señal que tiene solo unos pocos picos de nivel que desea controlar, pruebe con un valor de Relación de 4: 1 con un valor de Umbral alto para que solo se compriman los niveles más altos.
  • Para comprimir un cantante, comience con una proporción de 2: 1 (y un valor de umbral bajo) y aumente gradualmente el valor de proporción a su gusto.
  • Para suavizar un bajo o ajustar una caja, pruebe con un umbral medio / bajo y una proporción de 4: 1.
  • Para agregar sustain a una guitarra solista, pruebe con una proporción de 8: 1 con un umbral bajo

Una vez que tenga Threshold y Ratio funcionando como lo desea, use las otras configuraciones para ajustar su procesamiento de dinámica

5. AutoGain

AutoGain aumenta automáticamente las señales cuyos niveles se han reducido mediante el procesamiento dinámico para que alcancen un pico de -6 dB por debajo de 0 dB.

AutoGain establece la señal a un máximo de -6 dB para dejar un poco de margen para picos de volumen transitorios rápidos que ocurren antes de que el procesador dinámico tenga la oportunidad de responder.

6. Ataque

El parámetro Attack, que se puede establecer entre 0.0 ms (milisegundos) y 800.0 ms, establece la velocidad a la que comienza la reducción de ganancia cuando una señal:

  • supera el umbral cuando utiliza un compresor
  • cae por debajo del nivel de umbral cuando utiliza un expansor.

Si desea asegurarse de que su procesador dinámico capta incluso los picos o caídas más rápidos en la señal, acelere el ataque configurando Attack en un valor bajo. Si le preocupa más aplicar la reducción de ganancia a las señales que permanecen por encima o por debajo del umbral durante períodos de tiempo más largos, con compresión o expansión, respectivamente, configure Attack en un valor más alto y más largo.

Ajuste la configuración del parámetro Attack para ayudar a suavizar el bombeo audible que puede ocurrir con un valor de Ratio alto.

7. Liberar

El parámetro Release establece durante cuánto tiempo se aplicará la reducción de ganancia y se puede configurar de 0 a 8000 ms. Cuando su señal tenga muchos cambios rápidos de nivel, configure Release en un valor lo suficientemente alto como para que no escuche que el procesador de dinámica se enciende y apaga constantemente. Si solo está interesado en administrar algunos cambios de nivel aislados en una señal, pruebe con un valor más corto y más bajo.

Ajuste la configuración del parámetro Release para ayudar a suavizar el bombeo audible que puede ocurrir con un valor de Ratio alto.

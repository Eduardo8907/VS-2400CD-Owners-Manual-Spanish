Puede mantener presionado SHIFT mientras gira el dial Time / Value para moverse por los valores de los parámetros en incrementos finos en la pantalla CH EDIT VIEW.

1. ESTADO (solo canal de pista)

El parámetro STATUS refleja el comportamiento de los botones TRACK STATUS (Página 172). Cambiar el estado de grabación o reproducción de una pista en cualquier lugar lo cambia en ambos. Puede configurar el parámetro para:

  • REPRODUCIR: el botón TRACK STATUS del canal de la pista se ilumina en verde y cualquier dato que se haya grabado en la pista se reproduce cuando presiona el botón de transporte PLAY (Página 171 ).
  • REC — El botón TRACK STATUS parpadea en rojo para mostrar que la pista está ahora armada para grabación. Para saber cómo grabar, consulte “Grabación de una pista nueva” en la página 173.
  • MUTE: la luz TRACK STATUS se apaga y la pista se silencia.

2. V.Trk (solo canal de seguimiento)

El selector V.Trk le permite seleccionar una de las 16 V-Tracks de la pista para grabar y / o reproducir. Para obtener más información sobre V-Tracks, consulte la página 85.

Puede utilizar este parámetro para seleccionar el V-Track deseado de dos formas. En ambos casos, comienza seleccionando el parámetro V.Trk. Entonces tú puedes:

  • gire el dial Time / Value para seleccionar la V-Track deseada.
  • presione el botón intermitente ENTER / YES para ver la ventana emergente V. Trk que muestra gráficamente las 16 V-Tracks de la pista. Cada V-Track aparece como una línea. Si la línea es delgada, no hay datos grabados en el V-Track; si es gruesa, los hay. Gire el dial Time / Value para seleccionar la V-Track deseada y presione ENTER / YES para completar el proceso y cerrar la ventana.

También puede seleccionar el V-Track de un canal de pista con el mouse utilizando el mapa V-Track de la pantalla de inicio. Consulte la página 119.

3. ChLink

Cuando se activa CHLink (“Enlace de canal”), el canal actual se enlaza con el canal contiguo. Si el canal actual es:

  • número impar: está vinculado al canal de número par a su derecha.
  • número par: está vinculado al canal impar a su izquierda.

Cuando los canales están vinculados, puede cambiar el valor de un parámetro CH EDIT en ambos canales a la vez cambiándolo en cualquiera de los canales vinculados. Esto puede resultar especialmente útil cuando dos canales controlan los lados izquierdo y derecho de una señal estéreo.

Cuando enlaza canales de entrada o de pista, cada par de entradas o pistas resultante actúa como un único objeto estéreo durante el enrutamiento, con un único punto de conexión de entrada y salida.

Puede vincular canales rápidamente presionando el botón CH EDIT del canal par mientras mantiene presionado el botón CH EDIT para su socio impar.

Cuando dos canales están vinculados, sus parámetros PAN y FADER cambian de apariencia y se comportan de manera diferente. Consulte la página 152 y la página 150, respectivamente, para obtener más información.

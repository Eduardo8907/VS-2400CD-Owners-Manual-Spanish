Menú emergente de canales

Si está usando un mouse, también puede seleccionar una nueva entrada, pista o canal de retorno de efectos desde cualquier pantalla en la que se muestre el número del canal actualmente seleccionado como se muestra aquí.

Para seleccionar un nuevo canal:

  • Haga clic en el número de canal. Aparece el menú emergente de canales.
  • Mueva el cursor al canal deseado y haga clic.

Usando un teclado ASCII

Después de conectar un teclado ASCII a la toma de TECLADO PS / 2 y activarlo Página 64, puede utilizar el teclado para realizar las siguientes acciones.

Qué puede hacer con un teclado ASCII

Puede utilizar el teclado: A:
Para: teclas de letras y números ingrese nombres para proyectos, pistas, escenas, localizadores, etc.
Barra espaciadora iniciar, detener y continuar la reproducción del proyecto
Teclas de flecha mueva el cursor en la pantalla del VS-2400CD
MAYÚS + teclas de flecha acercar y alejar vertical y horizontalmente
Casa mostrar la pantalla de inicio
F7 mostrar la pantalla EZ Routing VIEW
SHIFT+F7 mostrar la pantalla EZ Routing P.BAY
SHIFT+F8 mostrar la pantalla Mastering Room
Ctrl+S almacenar el proyecto actual
Ctrl+Alt+Delete apague el VS-2400CD
F1-F6 Botones F1-F6
Tab el botón PAGE
Enter el botón ENTER / YES o el botón izquierdo del mouse
Esc el botón EXIT / NO
F9 el botón MENÚ PROYECTO
F10 el botón TRACK MENU
F11 el botón EFFECT MENU
F12 el botón UTILITY MENU

DESHACER y REHACER

Deshacer

La función Deshacer le permite invertir o "deshacer" las operaciones de grabación y edición de pistas. Cada proyecto recuerda hasta 999 de sus acciones de edición y grabación más recientes y puede restaurar el proyecto al estado en el que estaba antes de cada acción.

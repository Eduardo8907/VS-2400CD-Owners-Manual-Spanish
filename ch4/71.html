A cada acción se le asigna un número "Deshacer nivel". Cuanto menor sea el número del nivel, más reciente será la acción. Todas las acciones que se pueden deshacer en un proyecto aparecen en la lista Deshacer junto con la fecha y la hora de cada acción.

Cuando deshace una operación de grabación o edición, también se deshace todo lo que hizo después de la operación original.

Deshacer afecta solo a las operaciones de grabación y edición de pistas; no deshace los cambios en los valores de los parámetros.

Si un proyecto contiene datos necesarios para deshacer acciones que está seguro de que nunca necesitará deshacer, puede realizar una optimización del proyecto para descartar los datos no deseados y reducir el tamaño del proyecto y el espacio que ocupa en su disco duro. Consulte la página 96.

Deshacer una operación de grabación o edición de pistas

  1. Presione UNDO • REDO
  2. Busque la acción deseada en la lista Deshacer.
  3. Gire el dial TIME VALUE para seleccionar el nivel Deshacer deseado; el nivel seleccionado tiene una flecha a su izquierda
  4. Presione ENTER / YES. La acción seleccionada, y todas las demás acciones de grabación y edición que tuvieron lugar después de ella, se deshacen.

Una vez completada la operación Deshacer, el botón UNDO • REDO permanece encendido para indicar que puede cancelar la operación Deshacer realizando un Rehacer.

Puede configurar el parámetro UNDO MESSAGE del menú UTILITY para que cuando presione UNDO, el VS-2400CD invierta instantáneamente la última operación sin mostrar la lista Undo - vea la página 358.

Rehacer

Si cambia de opinión sobre una acción, o una serie de acciones, que ha deshecho, puede realizar un "Rehacer" mientras el botón UNDO • REDO está encendido. Para hacer esto, mantenga presionada la tecla SHIFT y presione UNDO • REDO. Cuando aparezca el cuadro de diálogo Rehacer, presione ENTER / YES.

La capacidad de realizar una operación de rehacer dura solo hasta que guarde el proyecto a continuación.

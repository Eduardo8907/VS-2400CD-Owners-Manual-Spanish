La pantalla de nombre del proyecto proporciona algunos elementos adicionales, incluida información sobre el proyecto y un área en la que puede ingresar comentarios sobre el proyecto.

Introducción de un nombre nuevo con los controles del VS-2400CD

  1. Presione ← para seleccionar el primer carácter en el cuadro de nombre si aún no está seleccionado.
  2. Gire el dial de tiempo / valor para seleccionar el primer carácter deseado; aparece un cuadro alrededor del carácter actualmente seleccionado en el área debajo del cuadro de nombre.
  3. Presione → para seleccionar el siguiente carácter.
  4. Gire el dial de tiempo / valor para seleccionar el segundo carácter del nuevo nombre
  5. Repita los pasos 3 y 4 hasta que haya ingresado el nombre nuevo completo.
  6. Cuando haya terminado, presione F5 (OK).

Introducción de un nombre nuevo desde un teclado

  1. Presione ← para seleccionar el primer carácter en el cuadro de nombre si aún no está seleccionado.
  2. Escriba el nombre deseado.
  3. Presione F5 (OK).

Herramientas de nombres

Los botones F debajo de la pantalla de nombres proporcionan herramientas que tal vez desee utilizar al crear un nombre.

Introducción de números con el teclado numérico

En los cuadros de parámetros que contienen ubicaciones de tiempo del proyecto, puede ingresar el número deseado usando el teclado numérico del VS-2400CD. Para hacer esto:

  1. Presione NUMERICS. Se enciende — y ENTER / YES y EXIT / NO parpadean — para indicar que el teclado está activo.
  2. Presione ← o → para seleccionar la parte del número que desea ingresar.

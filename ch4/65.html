Configuración del reloj del VS-2400CD

El VS-2400CD marca la hora de sus grabaciones para que le resulte más fácil realizar un seguimiento de su trabajo. Antes de que pueda hacer esto, deberá configurar su reloj.

  1. Mantenga presionada la tecla MAYÚS y presione F4 • UTILIDAD
  2. Pulse el botón PAGE repetidamente, si es necesario, hasta que aparezca "FECHA" encima de F5.
  3. Presione F5 (FECHA). Aparece la pantalla FECHA / HORA.
  4. Si el parámetro FECHA no está seleccionado, use los botones ←, →, ↑ y ↓ para seleccionarlo.
  5. Si el mes aún no está seleccionado, presione ← para seleccionarlo.
  6. Gire el dial de tiempo / valor hasta que se muestre el mes actual.
  7. Presione → y gire el dial de tiempo / valor para elegir la fecha de hoy
  8. Presione → y gire el dial de tiempo / valor para elegir el año actual
  9. Presione → y gire el dial de tiempo / valor para elegir el ajuste de FORMATO DE FECHA deseado.
    Las fechas se pueden mostrar en la pantalla en cualquiera de los siguientes estilos:
    10. Este valor: Muestra fechas como:
    mm/dd/yyyy 07/12/2001
    dd/mm/yyyy 12/07/2001
    yyyy/mm/dd 2001/07/12
    MMM.dd ‘YY Jul. 12, ‘01
    dd MMM ‘YY 12 Jul ‘01
  10. Presione ↓ para seleccionar el parámetro TIME.
  11. Gire el dial Time / Value para seleccionar la hora actual.

    12. El VS-2400CD usa un tiempo de 24 horas: la medianoche es la hora 00, la 1 am es la 01 y la 1 pm es la 13.

  12. Presione y gire el dial de tiempo / valor para seleccionar los minutos y luego los segundos.
  13. Presione F5 (SET) para confirmar sus cambios
  14. Presione F6 (EXIT) para salir de la pantalla FECHA / HORA y finalizar el procedimiento.

Algunos conceptos fundamentales

Selección

Antes de que pueda cambiar algo en el VS-2400CD, primero debe seleccionarlo. En el caso de los controles físicos, es obvio: antes de poder mover un fader, debes agarrarlo con los dedos. Esta regla también se aplica a los elementos que se muestran en la pantalla del VS-2400CD. Cuando selecciona algo en la pantalla, le está diciendo al VS-2400CD que desea hacer algo con lo que ha seleccionado. Las dos formas más comunes de hacer una selección son usar los botones del cursor o hacer clic en el elemento deseado con el mouse. Una vez que haya realizado una selección, puede realizar el cambio deseado con la herramienta adecuada.

Cuando selecciona un elemento mostrado y luego otro, está moviendo el "cursor de selección".

El VS-2400CD le muestra lo que está seleccionado actualmente. A menudo, el elemento seleccionado se "resalta": sus colores se invierten. El texto, por ejemplo, se vuelve blanco con un fondo oscuro.

Cuando selecciona un cuadro en el que puede ingresar datos o elegir un valor diferente, el contorno del cuadro se oscurece.

Los botones ENTER / YES y EXIT / NO

Use los botones ENTER / YES o EXIT / NO debajo de para ejecutar o cancelar la operación actual, o para responder a una pregunta planteada en la pantalla.

Puede distinguir entre el botón ENTER / YES y el EXIT / NO simplemente tocando: NTER / YES es más grande

Los botones ENTER / YES y EXIT NO parpadean para indicar que puede presionar:

  • Botón ENTER / YES para finalizar, o presione EXIT / NO para cancelar, la operación actual.
  • ENTER / YES para responder "sí" a una pregunta mostrada, o EXIT / NO para responder "no".

El botón ENTER / YES también parpadea para indicar que puede presionarlo para ver opciones adicionales. EXIT / NO parpadea a veces para mostrar que puede presionarlo para salir de la pantalla actual.

Si lo prefiere, siempre puede usar la tecla Enter en su teclado en lugar de ENTER / YES, o la tecla Esc en lugar de EXIT / NO.

El botón SHIFT

Muchos de los botones del VS-2400CD realizan más de un trabajo; estos botones a menudo tienen una etiqueta superior y una etiqueta inferior impresas dentro de un cuadro de contorno. La etiqueta sin caja muestra la función principal del botón, la etiqueta en caja su uso secundario.

Para hacer que un botón realice su trabajo secundario, mantenga presionado SHIFT y presione el botón antes de soltar SHIFT.

Puede configurar el botón SHIFT para que se active y desactive con cada toque para reducir las pulsaciones del botón SHIFT. Consulte “BLOQUEO DE MAYÚS” en la página 357.

Usando un mouse

El VS-2400CD utiliza un mouse de dos botones que se puede configurar para usuarios diestros o zurdos, de acuerdo con la configuración del parámetro MOUSE BUTTON SWAP Sw System en el menú UTILITY del VS-2400CD. Cuando se cambia el parámetro:

  • apagado: el mouse funciona como un mouse para diestros. El botón izquierdo hace clic y el botón derecho muestra menús emergentes.
  • encendido: el mouse funciona como un mouse para zurdos. El botón derecho hace clic y el botón izquierdo muestra menús emergentes.

Cuando el Manual del propietario dice "hacer clic" con el mouse, los usuarios diestros deben hacer clic en el botón de la izquierda. Cuando dice "hacer clic con el botón derecho", deben hacer clic en el botón de la derecha. Si es un usuario zurdo del mouse, tendrá que invertir mentalmente estas instrucciones: "haga clic" en el botón de la derecha y "haga clic con el botón derecho" en el botón de la izquierda.

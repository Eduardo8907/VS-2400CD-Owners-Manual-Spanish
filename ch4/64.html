Configuración del VS-2400CD

Configuración del monitor, el mouse y el teclado VGA

Esta sección describe cómo configurar su mouse, teclado y monitor VGA. Sus ajustes se encuentran en el menú UTILIDAD. Para llegar allí:

  1. Mantenga presionado SHIFT y presione F4 • UTILITY debajo de la pantalla
  2. Pulse PAGE, si es necesario, hasta que aparezca "SYSTEM" encima de F1
  3. Presione F1 (SYSTEM) debajo de la pantalla.
  4. Presione F2 (Param2) para mostrar la configuración de VGA, y luego F3 (Param3) para mostrar la configuración del mouse y el teclado.
  5. Utilice los botones ←,→,↑ y ↓ según sea necesario para seleccionar cada ajuste (el contorno de un ajuste se oscurece cuando se selecciona) y gire el dial de tiempo / valor para elegir el valor deseado.

Si no tiene un monitor o teclado VGA, omita sus parámetros.

VGA Monitor Parameters

Parámetro: Qué establece: El escenario típico es:
SALIDA VGA Enciende o apaga la salida VGA En
FRECUENCIA DE ACTUALIZACIÓN Establece la velocidad a la que el monitor vuelve a dibujar la imagen Configuración recomendada por el fabricante
H. POSICIÓN Desplaza la imagen hacia la izquierda o hacia la derecha 0
POSICIÓN V. Mueve la imagen hacia arriba o hacia abajo 0
OBJETIVO DE OPERACIÓN Designa la pantalla LCD interna o un monitor VGA conectado como pantalla de funcionamiento principal ver página 381

Consulte la documentación de su monitor VGA para conocer la frecuencia de actualización recomendada para evitar una mala calidad de imagen y posibles daños al monitor.

Los parámetros VGA se describen con mayor detalle en la página 355

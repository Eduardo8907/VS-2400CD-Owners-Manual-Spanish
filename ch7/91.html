Operaciones de proyecto

Acerca de F6 (MARK)

Antes de que pueda realizar algunas de las operaciones del proyecto, debe apuntar al proyecto en el que desea realizar la operación. Haz esto marcando el proyecto.

Marcar un proyecto

  1. Presione ↓ o ↑ gire el dial de tiempo / valor para seleccionar el proyecto

F6 (MARCA) está disponible solo cuando se selecciona un proyecto, no una unidad.

SELECCIONE

Presione SELECCIONAR para cargar el proyecto que está resaltado actualmente en la lista de proyectos. Cuando se carga, reemplaza el proyecto que está actualmente almacenado en la memoria integrada del VS-2400CD. Puede reproducir, grabar, editar y mezclar un proyecto solo mientras está cargado en el VS-2400CD.

El botón SELECT no se parece mucho a ningún otro botón de menú del VS-2400CD, y también actúa de manera diferente. Es el único botón que realiza su acción tan pronto como lo presiona. Todos los demás botones simplemente muestran las pantallas de operación correspondientes.

Para cargar un proyecto desde otra unidad, seleccione la unidad y presione F6 (LISTA), como se describe en “Para mostrar los proyectos en una unidad” en la página 90.

Cargando un proyecto

  1. Mantenga presionada la tecla MAYÚS y presione F1 (PROYECTO)
  2. Si "SELECCIONAR" no está visible sobre F1, presione PAGE hasta que lo esté.
  3. En la lista de proyectos, seleccione el proyecto que desea cargar.
  4. Presione F1 (SELECCIONAR). El VS-2400CD muestra:
  5. Presione ENTER / YES para cargar el proyecto seleccionado, o EXIT / NO para cancelar la operación. Si presionó ENTER / YES, el VS-2400CD pregunta:

    6. Si el proyecto cargado actualmente está protegido contra escritura (Página 95 ), el VS-2400CD carga el nuevo proyecto sin mostrar esta pregunta.

  6. Presione ENTER / YES para almacenar sus últimos cambios en el proyecto cargado actualmente antes de cargar el nuevo, o EXIT / NO para omitir volver a guardarlo. El VS-2400CD carga el proyecto seleccionado en su memoria

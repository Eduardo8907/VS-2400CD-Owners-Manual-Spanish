Seleccionar un elemento en la lista de proyectos

Para seleccionar un proyecto o unidad que se muestra en la lista de proyectos:

  1. Presione ↓ o ↑ gire el dial de tiempo / valor hasta que se destaque el elemento deseado.

Cuando un elemento está delineado y tiene una flecha apuntando hacia él desde la derecha, está seleccionado.

Para mostrar los proyectos en una unidad

  1. Seleccione la unidad. Cuando se selecciona una unidad, el botón LIST aparece en la esquina inferior derecha de la pantalla.
  2. Presione LIST — el VS-2400CD muestra una lista de los proyectos en la unidad.

Para ver el contenido de un CD de audio, consulte “La función del reproductor de CD” en la página 345.

Acerca de "Store Current?" Mensajes

Durante el transcurso de la mayoría de las operaciones del menú PROYECTO, el VS-2400CD le pregunta:

El VS-2400CD muestra este mensaje cuando una operación necesita tomar prestada temporalmente la parte de la memoria del VS-2400CD que contiene el proyecto cargado actualmente; una vez que se completa la operación, el VS-2400CD vuelve a cargar el proyecto desde su disco duro. Si ha realizado cambios en el proyecto desde la última vez que lo almacenó, presione ENTRAR / SÍ en respuesta a este mensaje para asegurarse de que esos cambios no se pierdan.

Si un proyecto está protegido contra escritura (Página 95 ), este mensaje no aparece, ya que el proyecto está bloqueado para evitar que se realicen cambios en él.

DIVISIÓN

SPLIT crea un nuevo proyecto a partir de V-Tracks seleccionados y los copia del proyecto cargado actualmente. El nuevo proyecto hereda la configuración del proyecto original, incluidos marcadores, localizadores, datos de Automix, el mapa de tempo y la pista de sincronización, la configuración del menú UTILITY relacionada con el proyecto y su nombre, con un último carácter distintivo agregado. El nuevo proyecto se almacena en la misma unidad que el proyecto original.

Project SPLIT puede ser útil si tiene un proyecto grande con muchas V-Tracks que no está usando pero que no quiere borrar. Puede dividir estas V-Tracks en un nuevo proyecto, haciendo que el proyecto original sea más pequeño y más fácil de respaldar.

SPLIT no se puede revertir usando la función Deshacer del VS-2400CD; las V-Tracks eliminadas de un proyecto pueden volver a él solo utilizando la función IMPORT de región. Ver página 267

Project SPLIT utiliza muy poco espacio adicional en su disco duro, ya que esencialmente mueve datos del proyecto original al nuevo, no crea nuevos datos.

División de un proyecto

  1. Cargue el proyecto que desea dividir (Página 91 )
  2. Mantenga presionado SHIFT y presione F1 (PROJECT).
  3. Si "SPLIT" no está visible encima de F3, presione PAGE hasta que lo esté.
  4. Presione F3 (SPLIT): aparece la primera pantalla SPLIT del VS-2400CD.

Puede marcar V-Tracks en la pantalla V-Track MAP de SPLIT, que se muestra arriba, o en su pantalla Select Track. Puede marcar tantas V-Tracks como desee en cualquier pantalla.

En el mapa de V-Tracks del proyecto, cualquier V-Track que contenga datos aparece como una pequeña caja negra con una casilla de verificación a su izquierda.

  1. Establezca el formato físico en:
    • Desactivado (Rápido): para un formateo más rápido si está formateando un disco duro VS-2400CD solo para borrarlo rápidamente.
    • Activado: si se formatea el disco duro para el VS-2400CD por primera vez o durante el mantenimiento mensual. Si tiene tiempo, el formato físico amplía considerablemente el proceso de formateo (40 minutos o más), utilice el formato físico para eliminar por completo cualquier corrupción de datos persistente
  2. Establecer Surface Scan en:
    • Desactivado: para omitir una comprobación de las superficies de los discos de disco duro durante el formateo.
    • Activado: para que compruebe las superficies de los discos del disco duro realizando una prueba automática de lectura / escritura después del formateo. Esto puede llevar un tiempo (30 minutos o más), pero es bueno hacerlo periódicamente.
  3. Presione F5 (ON). El VS-2400CD le pregunta si está seguro de que desea formatear el disco duro:
  4. Presione ENTER / YES para continuar con el formateo, o EXIT / NO para cancelar. Si presionó ENTER / YES, el VS-2400CD verifica dos veces para asegurarse de que desea continuar, ya que está a punto de borrar todo lo que está en el disco duro seleccionado.
  5. Presione ENTER / YES para continuar con el formateo, o EXIT / NO para cancelar la operación. Si presionó ENTER / YES, el VS-2400CD le preguntará si desea guardar los cambios recientes en el proyecto cargado actualmente.
  6. Presione ENTER / YES para guardar el proyecto, o EXIT / NO para continuar sin hacerlo. Dependiendo de su configuración de formato físico y escaneo de superficie, el formateo puede demorar desde unos momentos hasta varias horas.
No interrumpa la operación de formateo antes de que finalice.

Partición clara

Puede borrar o "inicializar" el contenido de una sola unidad (una partición) para dejar espacio para nuevos proyectos. La operación Borrar partición del menú PROYECTO realiza esta tarea.

Borrar una unidad no tiene ningún efecto sobre la fragmentación. Solo hacer una copia de seguridad de sus proyectos y reformatear un disco duro puede fragmentarlo.

Borrar una unidad / partición

  1. Mantenga presionada la tecla MAYÚS y presione F1 (PROYECTO)
  2. Seleccione la unidad que desea borrar.
  3. Si "ClrPrt" no está visible encima de F3, presione PAGE hasta que lo esté.

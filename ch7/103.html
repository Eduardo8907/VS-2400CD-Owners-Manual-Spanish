RECUPERAR

Para volver a cargar los datos de respaldo del proyecto VS-2400CD, utilice la operación RECUPERAR. Cuando recupera un proyecto, el proyecto se restaura en su disco duro interno en su estado original.

RECUPERAR está disponible solo cuando la unidad seleccionada contiene datos de respaldo del proyecto.

Recuperación de datos de respaldo

  1. Mantenga presionado SHIFT y presione F1 (PROJECT).
  2. Inserte el CD que contiene la copia de seguridad en su unidad de CD-R / RW.

    3. Si se realiza una copia de seguridad del proyecto en más de un disco, inserte el primer disco que contenga los datos de la copia de seguridad.

  3. Seleccione la unidad que contiene los datos de la copia de seguridad del proyecto.
  4. Press F6 (LIST) to view the drive’s contents. The VS-2400CD displays:
  5. Presione ENTER / YES para continuar, o EXIT / NO para cancelar el cambio de unidad.
  6. Marque los proyectos que desea recuperar.
  7. Gire el dial de tiempo / valor para seleccionar la unidad en la que desea recuperar los datos de copia de seguridad marcados.
  8. Si "Recovr" no está visible encima de F2, presione PAGE hasta que lo esté.
  9. Presione F2 (Recovr); aparece la pantalla PROJECT RECOVER.
  10. Configure Borrar todos los proyectos en:
    • Desactivado: si desea agregar los proyectos que está recuperando a la lista de proyectos que ya se encuentran en la unidad de destino.
    • Activado: si desea borrar la unidad para que solo contenga sus proyectos recuperados
  11. Configure la velocidad de la unidad de CD deseada. Prueba MAX. Si tiene problemas para recuperar los datos, pruebe con una velocidad más baja.
  12. Presione F5 (ON). El VS-2400CD le pregunta si desea guardar el proyecto cargado actualmente.
  13. Presione ENTER / YES para guardar el proyecto cargado antes de la operación de recuperación, o presione EXIT / NO para ir directamente a la recuperación. El VS-2400CD le pregunta si está seguro de que desea continuar con la recuperación.
  14. Presione ENTER / YES para continuar con la recuperación, o EXIT / NO para cancelar.
  15. Si Borrar todos los proyectos está activado, el VS-2400CD le pregunta si está seguro de que desea borrar primero la unidad de destino; presione ENTER / YES para hacerlo, o EXIT / NO para cancelar la recuperación.
  16. Si la copia de seguridad es más grande que un solo disco, el VS-2400CD solicita discos adicionales según sea necesario. Inserte cada disco solicitado en su unidad y presione ENTER / YES para continuar.

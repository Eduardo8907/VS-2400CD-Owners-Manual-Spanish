Crear un nuevo proyecto

  1. Mantenga presionado SHIFT y presione F1 (PROJECT)
  2. Si "NUEVO" no está visible encima de F2, presione PAGE hasta que lo esté.
  3. Presione F2 (NUEVO): aparece la pantalla PROYECTO NUEVO.
  4. Configure el parámetro Copy System Prm en:
    • Desactivado: para crear un nuevo proyecto con los valores predeterminados de los parámetros del menú UTILITY (los parámetros del menú UTILITY se describen en el Capítulo 27 ).
    • Activado: para copiar los valores de los parámetros UTILITY relacionados con el proyecto actual en el nuevo proyecto.
  5. Configure el parámetro Copy Mixer / Scene Prm en:
    • Desactivado: para crear un nuevo proyecto con la configuración predeterminada del mezclador y sin escenas.
    • Activado: para copiar la configuración del mezclador y las escenas del proyecto cargado actualmente en el nuevo proyecto.

    6. Dado que las escenas recuerdan las ediciones de parches de efectos, copiar escenas le permite mover la configuración de efectos del proyecto actual al nuevo que está creando.

  6. Select the desired sample rate and recording mode (Page 92 ).
  7. Nombra el nuevo proyecto.

    8. Puede cambiar el nombre del proyecto más adelante, y agregar un comentario, si lo desea, mediante la operación NOMBRE del proyecto (consulte la página 95 ).

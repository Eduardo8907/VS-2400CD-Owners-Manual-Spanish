IMPORTAR

Puede utilizar el proyecto IMPORT para restaurar la copia de seguridad de la canción y los datos de archivo desde:

  • VS-880
  • VS-880EX
  • VSR-880
  • VS-890
  • VS-1680EX
  • VSR-1880
  • VS-2480/VS-2480CD

Cuando importa una canción de un V-Studio anterior, el VS-2400 la convierte en un proyecto VS-2400. Puede recuperar datos de canciones respaldados desde un CD. Una vez que el VS-2400 importa la canción, se carga como proyecto actual. Si está importando varias canciones, la última canción que importa el VS-2400 se convierte en el proyecto actual.

Consulte “Compatibilidad de proyectos de V-Studio Song / VS-2400CD” en la página 468 para obtener detalles sobre cómo las canciones de otros V-Studios se transforman en proyectos del VS-2400CD.

No puede importar canciones desde el VS-1680 o VS-1880 instalando sus discos duros internos en el VS-2400CD. El VS-2400CD utiliza un tipo diferente de formato de unidad.

Para importar una canción del VS-840, utilice la operación de conversión de canción del VS-840 para convertirla primero al formato VS-880 y luego importe la canción al VS-2400CD. Solo se importan las asignaciones de pistas y de audio de la canción; todos los demás ajustes deben volver a crearse manualmente

Importar una canción desde un V-Studio anterior

  1. Inserte el CD que contiene los datos de la canción respaldados en su unidad de CD-R / RW.
  2. Mantenga presionado SHIFT y presione F1 • PROJECT.
  3. Seleccione el dispositivo que contiene los datos de la canción.
  4. Presione F6 (LIST). El VS-2400CD muestra:
  5. Presione ENTER / YES para cambiar de unidad, o EXIT / NO para cancelar el cambio de unidad.
  6. Marque la (s) canción (s) que desea importar
  7. Seleccione la unidad a la que desea importar la canción.
  8. Si "IMPORT" no está visible encima de F3, presione PAGE hasta que lo esté.
  9. Presione F3 (IMPORT); aparece la pantalla PROJECT IMPORT.
  10. . Presione F5 (OK) para importar las canciones marcadas; el VS-2400CD le preguntará si está seguro.
  11. Presione ENTER / YES para comenzar a importar, o EXIT / NO para cancelar la operación. El VS-2400CD importa los datos de la canción seleccionada. Dependiendo de la cantidad de datos y la cantidad de conversión requerida, esto puede llevar un tiempo.

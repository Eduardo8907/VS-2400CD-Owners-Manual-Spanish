EXPORTAR

Puede exportar el proyecto actual desde el VS-2400CD para que pueda reproducirse en:

  • VS-880
  • VS-880EX
  • VSR-880
  • VS-890
  • VS-1680EX
  • VSR-1880
  • VS-2480/VS-2480CD

El VS-2400CD no exportará un proyecto directamente al formato de datos del VS-840. Puede exportar un proyecto como datos del VS-880 y utilizar la función de conversión de canciones del VS-840 para cargar los datos.

Cómo funciona la exportación

Esta sección describe cómo se traduce el proyecto VS-2400CD cargado actualmente para exportarlo a los diversos V-Studios. En todos los casos, cuando exporta un proyecto VS-2400CD, la canción recién creada utiliza la frecuencia de muestreo y el modo de grabación del proyecto.

El modo de grabación M16 del VS-2400CD es el mismo que el modo MAS en el VS-880, VS-880EX, VSR-880, VS-890, VS-1680 y VS-1880. El modo de grabación M16 del VS-2400CD es idéntico al del VS-2480 y VS-2480CD.

Puede exportar un proyecto a CD-R / RW como un archivo de canciones para V-Studios anteriores y como copia de seguridad para el VS-2480 / VS-2480CD.

Exportar a un VS-880 o VS-880EX
  • Se exportan ocho pistas de proyecto. Puede seleccionar las pistas 1-8, 9-16 o 17-24.
  • Solo los V-Tracks 1-8 se exportan a las canciones del VS-880; para exportar datos en V-Tracks 9-16, copie los datos en V-Tracks 1-8. En las canciones del VS-880EX, las V-Tracks 1-8 del VS-2400CD se exportan al banco A y las V-Tracks 9-16 al banco B.
  • Solo se exportan los localizadores 00-31 del proyecto.
  • Los proyectos que utilizan los modos de grabación M24, MTP, CDR y LV2 no se pueden exportar como canciones VS-880 o VS-880EX.
Exportar a un VS-890 o VSR-880
  • Se exportan ocho pistas de proyecto. Puede seleccionar las pistas 1-8, 9-16 o 17-24
  • Las V-Tracks 1-8 se exportan al Banco A y las V-Tracks 9-16 al Banco B
  • Los proyectos que utilizan los modos de grabación M24 y LV2 no se pueden exportar como canciones VS-890 o VSR-880.
  • Los datos del proyecto grabados en MTP se convierten al modo VSR durante la exportación; este proceso puede llevar un tiempo
Exportando a una VS-1680
  • Se exportan 16 pistas de proyecto. Puede seleccionar las pistas 1-16 o 9-24.
  • Solo se exportan los localizadores 00-63 del proyecto.
  • Los proyectos que utilizan los modos de grabación M24 y CDR no pueden exportarse como canciones del VS-1680, ni tampoco los proyectos que contienen pistas de masterización CDR; consulte la página 332
Exportando a una VS-1880
  • Se exportan 18 pistas de proyecto. Puede seleccionar las pistas 1-18 o 7-24.
  • Solo se exportan los localizadores 00-63 del proyecto.
  • Los proyectos que utilizan el modo de grabación M24 no se pueden exportar como canciones del VS-1880.

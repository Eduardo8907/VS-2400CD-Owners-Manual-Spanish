Hacer una copia de seguridad de un proyecto

  1. Inserte el CD-R o CD-RW deseado en su unidad de CD-R / RW.

    2. La operación BACKUP vuelve a formatear el medio de destino para su uso como copia de seguridad. Por lo tanto, no es necesario formatear los medios con anticipación.

  2. Mantenga presionado SHIFT y presione F1 (PROJECT).
  3. Marque los proyectos de los que desea realizar una copia de seguridad.
  4. Si "BACKUP" no está visible encima de F1, presione PAGE hasta que lo esté.
  5. Presione F1 (COPIA DE SEGURIDAD): aparece la pantalla COPIA DE SEGURIDAD DEL PROYECTO.
  6. Seleccione si desea o no que el VS-2400CD verifique la precisión de su copia de seguridad una vez que se haya escrito en el CD. Si bien esto hace que la copia de seguridad tarde un poco más, recomendamos activar Verificar para asegurarse de que los datos de su proyecto se hayan respaldado correctamente.
  7. Seleccione la velocidad de escritura de la grabadora de CD. El VS-2400CD le ofrece solo velocidades admitidas por la unidad. Recomendamos probar la configuración MAX que utiliza la velocidad admitida más rápida de su unidad. Si tiene algún problema, pruebe con una velocidad más baja.
  8. Presione F5 (OK). El VS-2400CD le pregunta si desea guardar los cambios recientes en el proyecto cargado actualmente antes de continuar con la copia de seguridad.
  9. Presione ENTER / YES para guardar su proyecto cargado actualmente antes de continuar con la copia de seguridad, o EXIT / NO para pasar directamente a la copia de seguridad. Si está utilizando un disco CD-RW que ya contiene datos, el VS-2400CD le pregunta si desea borrar el disco. Para obtener detalles sobre los mensajes que pueden aparecer, consulte la página 344.
  10. Presione ENTER / YES para borrar el CD-RW, o EXIT / NO para cancelar la copia de seguridad. Si un proyecto es demasiado grande para caber en el medio que está utilizando, el VS-2400CD le solicita medios adicionales según sea necesario.

    11. Durante una copia de seguridad de varios CD, el VS-2400CD solicita cada disco dos veces: una para etiquetar el disco y otra para escribir datos en el disco.

  11. . Inserte el medio solicitado y presione ENTER / YES, o presione EXIT / NO para cancelar el procedimiento.
  12. Si está realizando una copia de seguridad de varios proyectos, el VS-2400CD realiza una copia de seguridad de cada uno de ellos.

Los datos de la copia de seguridad del proyecto no se pueden reproducir tal cual. Para reproducir el proyecto, use Project RECOVER para restaurar el proyecto a su disco duro en forma reproducible

  1. Presione F3 (ClrPrt); aparece.
  2. Establecer Surface Scan en:
    • • Desactivado: para omitir una comprobación de la superficie del plato en el área de la partición del disco duro durante la inicialización.
    • Activado: para que compruebe la superficie del plato realizando una prueba automática de lectura / escritura después de la inicialización. Esto puede llevar un tiempo, pero es bueno hacerlo periódicamente.
  3. Presione F5 (ON). El VS-2400CD le pregunta si está seguro de que desea borrar la unidad seleccionada:
  4. Presione ENTER / YES para borrar la unidad o EXIT / NO para cancelar. Si presionó ENTER / YES, el VS-2400CD verifica dos veces para asegurarse de que desea continuar, ya que está a punto de borrar todo en la unidad / partición seleccionada.
  5. Si desea continuar, presione ENTER / YES. Para cancelar la operación, presione EXIT / NO.
  6. Si presionó ENTER / YES, el VS-2400CD le preguntará si desea guardar los cambios recientes en el proyecto cargado actualmente.
  7. Presione ENTER / YES para guardar el proyecto antes de borrar la unidad, o EXIT / NO para continuar sin hacerlo.

Comprobación de conducción

Si tiene problemas para cargar un proyecto, copiarlo o realizar una copia de seguridad, puede ejecutar una operación de Verificación de unidad para determinar si el problema está en sus datos; de lo contrario, es posible que tenga un problema de conexión o un mal funcionamiento del hardware. Drive Check examina el directorio que realiza un seguimiento de dónde están sus archivos y busca problemas potenciales con sus datos. Cuando termina, muestra un informe de sus hallazgos.

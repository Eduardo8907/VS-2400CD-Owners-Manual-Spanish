Cambiar su ubicación actual en un proyecto

Puede cambiar su ubicación usando la visualización de ubicación de tiempo actual, moviéndose a través del proyecto en cualquiera de sus incrementos de tiempo.

Usar el: Hacer esto:
ratón 1. Haga clic en la unidad de medida de tiempo deseada y arrastre hacia arriba o hacia abajo para avanzar o retroceder en el proyecto.
Dial de tiempo / valor
  1. Utilice o para subrayar la unidad de medida de tiempo deseada.
  2. Gire el dial de tiempo / valor en sentido horario o antihorario para avanzar o retroceder en el proyecto. Mantenga presionada la tecla MAYÚS mientras gira el dial para moverse por períodos de tiempo más largos.

Puede moverse a través de un proyecto en 1/100 de fotograma seleccionando el valor de subcuadro del contador de código de tiempo y manteniendo presionada la tecla MAYÚS mientras gira el dial de tiempo / valor.

Reloj, Calendario

El reloj muestra la hora actual y el calendario muestra la fecha de hoy.

Para que el calendario y el reloj muestren con precisión la fecha y la hora actuales, debe configurar el reloj del sistema del VS-2400CD. Consulte la página 65.

Puede reemplazar el calendario y el reloj con una lectura que le muestre cuánto espacio libre queda en la unidad seleccionada. Consulte “DATE / REMAIN Sw” en la página 359.

Uso de la pantalla Fader / Pan

Puede reemplazar la visualización de medidores (Página 116 ) en la pantalla de inicio con la visualización de fader / pan si lo desea. La pantalla de fader / pan presenta una pequeña tira de canal que representa cada canal del tipo seleccionado con los interruptores F / P. Cada banda de canal en pantalla contiene un fader y un control de panorámica que puede configurar con el mouse.

Los interruptores F / P

Los interruptores F / P activan la visualización de fader / pan y seleccionan el conjunto de canales que le gustaría ver. Para mostrar los interruptores F / P, presione PAGE hasta que aparezca “IN F / P” encima de F2.

F5 (FX RTN) muestra los medidores para los cuatro canales de retorno FX, como se describe en Página 121.

Indicadores de pico de entrada

Los indicadores de pico de entrada le ayudan a establecer el nivel de las señales que ingresan a las tomas de entrada analógica del VS-2400CD. Los 16 indicadores numerados corresponden a las 8 tomas de entrada analógica y las ocho conexiones de entrada digital R-BUS, respectivamente.

Cuando está ajustando una perilla SENS (Página 34) para establecer el nivel de entrada para una señal que ingresa a una toma de entrada analógica, aparece un cuadro oscuro alrededor de su indicador de pico de entrada si la señal excede un nivel predeterminado: INPUT PEAK LEVEL (Página 356 ) configura los indicadores de pico de entrada para que se iluminen cuando una señal alcanza -6 dB, -3 dB o 0 dB. Aparece una línea gris claro en la vista de pantalla de los medidores INPUT (Página 120 ) para mostrar el ajuste del umbral de volumen actual de los indicadores de pico de entrada.

Visualización de ubicación de hora actual

En la parte superior de muchas de las pantallas del VS-2400CD se encuentra la visualización de la ubicación del tiempo actual, que consta de tres contadores de tiempo del proyecto. Cada uno le muestra la ubicación actual de la línea ahora en el proyecto utilizando su propia forma de medición del tiempo:

Este contador: Muestra la hora actual:
Código de tiempo como horas, minutos, segundos, fotogramas y subtramas
Medidas y latidos como compases, tiempos y tics (hay 480 tics en cada latido)
Marcadores mostrando el último marcador que pasó por la línea ahora

Si está grabando instrumentos MIDI desde un secuenciador sincronizado con el VS-2400CD, el contador de compases y tiempos puede facilitar la realización de ediciones de pista muy precisas en el VS-2400CD. Asegúrese de configurar el ritmo y el tempo del proyecto para que coincidan con los del secuenciador antes de grabar. Consulte “Trabajar con un mapa de tempo” en la página 298.

Marco, subchasis, tick

El indicador ABS / REL le dice si el tiempo mostrado es:

  • tiempo absoluto (ABS): basado en la hora de inicio real del proyecto.
  • tiempo relativo (REL): se cambia durante la sincronización del VS-2400CD con un dispositivo externo. Consulte “Cambio de la hora de inicio del proyecto” en la página 303.
